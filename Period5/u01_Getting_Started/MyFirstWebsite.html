HTML has two main types of elements:

  1. Block elements: Automatically inserts line breaks before and after the element.
  2. Inline elements: An example of an inline element is the <em> tag.

A few observations...

  • You can "nest" inline elements inside of block elements.
  • For example, you could nest an em tag inside of a p tag: <p>This is <em>emphasized</em></p>
  • Do NOT nest a block level element inside an inline element.
  • For example, do NOT nest a p tag inside an em tag: <em>This is <p>emphasized</p></em>
