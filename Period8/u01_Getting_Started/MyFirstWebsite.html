HTML has two main types of elements:

  1. Block elements: Automatically inserts line breaks before and after the element.
  2. Inline elements: An example of an inline element is the <em> tag.

Some observations on block/inline elements

  • You can nest inline elements inside of block elements.
  • For example: <p>This text is <em>important</em></p>.
  • You can NOT nest block elements inside inline elements.
  • For an anti-example: <em>Putting a <p>block tag</p> inside is bad.</em>
